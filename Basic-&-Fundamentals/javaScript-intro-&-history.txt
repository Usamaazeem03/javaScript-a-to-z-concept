🟨 JAVASCRIPT INTRODUCTION
JavaScript is a high-level, cross-platform, object-oriented scripting language.  
It is mainly used to make webpages interactive (e.g., clickable buttons, animations, forms, etc.).  

✨ Without JavaScript, websites would be static (just HTML + CSS).  


📜 JAVASCRIPT HISTORY
A genius named Brendan Eich created JavaScript at Netscape in just 10 days!  
- First name: Mocha ☕  
- Then: LiveScript 🔴  
- Finally: JavaScript 🟨  


🕰️ TIMELINE

1995 : Birth of JavaScript (Netscape) 🍼  
1997 : ECMAScript 1 (ES1) 📖  
1999 : ECMAScript 3 (ES3) ⚡  
2005 : AJAX (Dynamic Web Apps) 🔄  
2006 : jQuery ✨  
2008 : Google V8 Engine 🚀  
2009 : ES5 + Node.js 🌍  
2013 : React ⚛️  
2015 : ES2015 (ES6) 🔥 (let, const, arrow functions, classes)  
2017 : async/await, WebAssembly 1.0 ⏳🧩  
2018-2021 : Optional chaining (?.), nullish coalescing (??), modern JS ✨  
2020 : Deno 🦕  
2022+ : Bun, Vite, esbuild, SWC ⚡⚡
